# FinTrack - Personal Finance Tracker

## Description
A web app to track income, expenses, and savings goals. Features include visual charts, budget categories, and monthly reports.

## Features
- Add and categorize income/expenses
- Visualize spending with charts
- Set and track savings goals
- Export monthly reports (CSV)

## Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js (for charts)

## Setup
1. Clone the repository.
2. Open `index.html` in your browser.

## Responsive Design
The app is fully responsive for mobile and desktop.
