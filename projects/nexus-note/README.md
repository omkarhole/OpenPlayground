# Nexus Notes - Networked Visual Note-Taking App

Nexus Notes is a unique, browser-based note-taking application that allows you to create, connect, and visualize notes in an interactive graph. Unlike traditional linear note-taking apps, Nexus Notes helps you see relationships between ideas by creating a visual knowledge network.

## Features

- **Visual Note Creation**: Create notes with titles, content, tags, and color coding
- **Interactive Graph View**: Visualize your notes as connected nodes in a force-directed graph
- **Note Connections**: Link related notes together to build knowledge networks
- **Multiple Note Types**: Categorize notes as Ideas, Tasks, References, or Questions
- **Local Storage**: All notes are saved locally in your browser
- **Tag System**: Organize notes with customizable tags
- **Search Functionality**: Quickly find notes by title, content, or tags
- **Responsive Design**: Works on desktop and mobile devices

## How to Use

1. **Create a Note**: Click the "New Note" button and fill in the details
2. **Edit Notes**: Click any note in the sidebar to edit its content
3. **Add Tags**: Type a tag in the tag input and press Enter
4. **View Graph**: Click the "Graph View" button to see your notes as a network
5. **Create Connections**: In graph view, double-click a node to start a connection, then double-click another node to complete it
6. **Remove Connections**: Click the unlink icon in the connections list or in graph view

## Technical Details

- Built with vanilla HTML, CSS, and JavaScript
- Uses D3.js for the interactive graph visualization
- Stores data locally using browser's localStorage API
- Responsive design using CSS Flexbox and Grid
- No external dependencies except D3.js and Font Awesome icons
