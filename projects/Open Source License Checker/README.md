# Open Source License Checker

A utility that scans project files and dependencies to detect and display open source licenses. It helps developers ensure compliance and understand the legal aspects of their codebase.

## Features
- Scan project files and dependencies for license information
- Display detected licenses in a summary table
- Highlight incompatible or missing licenses
- Export license report as CSV
- Filter by license type or file
- Link to license texts and explanations
- Responsive, works in dark and light mode

## Usage
1. Open `index.html` in your browser.
2. (Demo only) Interact with the sample license table and UI.
3. For real license scanning, integrate a backend or WASM parser.

## Tech
- Vanilla HTML, CSS, JavaScript

## Responsive
- Fully responsive for desktop and mobile.
