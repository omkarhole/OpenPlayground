# Alien Abduction

An 8-bit style HTML5 arcade game where you pilot a UFO to abduct farm animals while avoiding enemies and managing your ship's energy.

## How to Play

- **Controls:**
  - WASD or Arrow Keys: Move the UFO
  - Space: Activate abduction beam
  - Shift: Boost speed
  - Gamepad supported

- **Objective:**
  - Abduct animals (Cows: 100pts, Pigs: 150pts, Hens: 200pts)
  - Chain captures for combo multipliers
  - Avoid farmers and tanks
  - Manage your HP and Energy levels

- **Game Features:**
  - Multiple biomes (Farm, Desert, Snow, Wasteland)
  - Retro 8-bit graphics and sound
  - High score system
  - Mobile touch controls

## Files

- `index.html`: Main game HTML file
- `app.js`: Game logic and rendering
- `style.css`: Game styling and UI

## Running the Game

Simply open `index.html` in a modern web browser. No additional setup required.

## Technologies Used

- HTML5 Canvas for rendering
- Vanilla JavaScript for game logic
- CSS for styling
