CSS Cheat Sheet

Selectors

Elements
Let's you style all of an HTML element on a page.
p {
  background-color: pink;
}
Classes
Let's you target any object with a matching class attribute.
.your-class {
  background-color: pink;
}
IDs
Let's you target any object with a matching id attribute.
#yourID {
  background-color: pink;
}

Properties

Background

background-color:
Add a solid background color to an object. Is usually set with a hexadecimal value, named color, or an rgb value.
background-color: salmon;
background-color: #FA8072;
background-color: rgb(250,128,114);
background-image:
Sets an image as a background to an object.
background-image: url(image.jpg);
background-attachment:
Sets how an elements background behaves when the page scrolls.
background-attachment: scroll; Default, background scrolls with the content.
background-attachment: fixed; Background remains stationary while content scrolls.
background-repeat:
Sets how a background image repeats.
background-repeat: repeat; Default, background repeats in both directions
background-repeat: repeat-x; Background repeats horizontally, along the x axis.
background-repeat: repeat-y; Background repeats vertically, along the y axis.
background-repeat: no-repeat; Background does not repeat.

Text

font-family:
Sets the display font of an elements text. Accepts a comma separated list of fonts. The first font will be loaded first and all others will be backup fonts. A font type is usually added at the end of the list. Fonts with spaces need to be wrapped in quotes.
font-family: 'Times New Roman', Georgia, serif;
font-size:
Sets the size of an element's text.
font-size: 16px;
font-size: 1em;
font-weight:
Sets the weight or thickness of a font.
font-weight: normal; Default
font-weight: bold;
font-style:
Allows you to make text appear italicized.
font-style: normal; Default
font-style: italic;
text-decoration:
Allows you to add an underline, overline, line-through, or a combination of lines to selected text.
text-decoration: none; Default
text-decoration: underline; Adds a line under text.
text-decoration: line-through; Adds a line through text.
text-decoration: overline; Adds a line above text.
color:
Allows you to change the color of text.
font-color: black;
font-color: #000000;
line-height:
Sets the amount of vertical space a line of text will use.
line-height: 16px;
line-height: 1em;

Border

border-width:
Sets the width of a border
border-width: 1px;
border-color:
Sets the color of a border
border-color: black;
border-style:
border-style: none; Default, no border.
border-style: solid; Adds a solid line.
border-style: dashed; Adds a dashed line.
border-style: dotted; Adds a dotted line.
border-radius:
Adds rounded corners to an element. Accepts pixel values, em values, and percentage.
border-radius: 16px;
border-radius: 1em;
border-radius: 10%;

Box-Model

display:
Allows you to display the default display of an element. Some are block elements and some are inline. Find a really good explanation at learnlayout.com/display.html.
display: none; Hides an element.
display: block; Fills the entire width of a space unless a set width is set. Responds to width, height, margin, and padding.
display: inline; Flows with other inline elements. Does not respond to width, height, margin, or padding.
display: inline-block; Flows with inline elements but also responds to width, height, margin, and padding.
float:
Used for positioning elements on a page.
float: none; Default.
float: left; Pushes an element to the left side of it's parent element.
float: right; Pushes an element to the right side of it's parent element.
clear:
Used with floats, clear can affect the other elements before or after your element.
clear: none; Default.
clear: left; Pushes all elements that would otherwise be on the left of your element to the space above.
clear: right; Pushes all elements that would otherwise be on the right of your element to the space below.
width:
Set's the amount of horizontal space an element will use.
width: 200px;
width: 10em;
width: 90%;
max-width:
Set's the maximum amount of horizontal space an element will use.
max-width: 200px;
max-width: 10em;
max-width: 90%;
min-width:
Set's the minimum amount of horizontal space an element will use.
min-width: 200px;
min-width: 10em;
min-width: 90%;
height:
Set's the amount of vertical space an element will use.
height: 200px;
height: 10em;
height: 90%; *Percentage heights will only work if a parent element has a set height (pixels or ems).
max-height:
Set's the maximum amount of vertical space an element will use.
max-height: 200px;
max-height: 10em;
max-height: 90%;
min-height:
Set's the minimum amount of vertical space an element will use.
min-height: 200px;
min-height: 10em;
min-height: 90%;
margin:
Adjust the space around all sides on the outside of an element.
margin: auto; Default, let's the browser calculate the margin.
margin: 20px;
margin: 2em;
margin-top:
margin-bottom:
margin-left:
margin-right:
Let's you adjust the ouside space on a specific side of an element.
padding:
Adjust the space around all sides on the inside of an element.
padding: 20px;
padding: 2em;
padding-top:
padding-bottom:
padding-left:
padding-right:
Let's you adjust the inside space on a specific side of an element.

Pseudo Classes

:hover
Applies styles to an element when the mouse cursor is currently over in.
a:hover {
  color: pink;
}
:first-child
Targets the first instance of an element in a container.
p:first-child {
  font-weight: bold;
}
:last-child
Targets the last instance of an element in a container.
p:last-child {
  font-style: italic;
}

Flexbox

display: flex;
Turns an element into a flex container.
display: flex;
flex-direction:
Defines the direction of flex items.
flex-direction: row;
flex-direction: column;
justify-content:
Aligns items horizontally.
justify-content: center;
space-between, space-around
align-items:
Aligns items vertically.
align-items: center;
gap:
Adds space between flex items.
gap: 16px;

Position

position:
Controls how an element is positioned.
position: static; Default
relative, absolute, fixed, sticky
top / right / bottom / left
Moves positioned elements.
top: 10px;
z-index:
Controls stack order.
z-index: 10;

Overflow

overflow:
Controls content that overflows its container.
overflow: hidden;
scroll, auto
visibility:
Shows or hides an element without removing space.
visibility: hidden;

Animation

transition:
Adds smooth animation.
transition: all 0.3s ease;
transform:
Transforms elements.
transform: scale(1.1);
rotate(45deg), translateX(20px)
opacity:
Controls transparency.
opacity: 0.5;

p::before { content: "üëâ"; }

Common Shortcuts

center a div
margin: auto;
center with flex
display: flex;
justify-content: center;
align-items: center;
full screen height
height: 100vh;

Lists

list-style-type:
Changes bullet or number style.
list-style-type: disc;
circle, square, decimal, none
list-style-position:
Controls bullet position.
list-style-position: inside;
list-style:
Shortcut for list styles.
list-style: none;

Tables

border-collapse:
Merges table borders.
border-collapse: collapse;
border-spacing:
Space between table cells.
border-spacing: 10px;
table-layout:
Controls table width calculation.
table-layout: fixed;

Forms

outline:
Border shown on focus.
outline: none;
:focus
Applies styles when input is selected.
input:focus { border-color: blue; }
cursor:
Changes mouse cursor.
cursor: pointer;

Background (Advanced)

background-size:
Controls background image size.
background-size: cover;
contain, auto
background-position:
Sets image position.
background-position: center;

Shadows

box-shadow:
Adds shadow to elements.
box-shadow: 0 4px 10px rgba(0,0,0,0.2);
text-shadow:
Adds shadow to text.
text-shadow: 2px 2px 5px black;

Responsive

@media
Applies styles based on screen size.
@media (max-width: 768px) {
  body { font-size: 14px; }
}
vh / vw
Viewport height and width units.
width: 100vw;

More Pseudo Classes

:nth-child()
Selects elements by position.
li:nth-child(2) { color: red; }
:not()
Excludes elements.
p:not(.active) { color: gray; }
:checked
Targets checked inputs.
input:checked { accent-color: green; }

Best Practices

Use shorthand
margin: 10px 20px;
Avoid fixed heights
Use min-height instead.
Use class selectors
Avoid styling by ID.
