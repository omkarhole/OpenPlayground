## CSS Cheat Sheet

This project provides a quick reference guide for common CSS selectors, properties, pseudo-classes, flexbox, tables, forms, and more. It is designed to help developers and learners easily find and understand essential CSS syntax and usage examples.

### Usage
Open `index.html` in your browser to view the cheat sheet. Browse through the sections for instant CSS tips and code samples.

### Features
- Covers selectors, properties, box model, flexbox, tables, forms, shadows, responsive design, and best practices
- Includes code examples for each concept
- Easy to navigate and visually organized
