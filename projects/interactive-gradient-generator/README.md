# Gradient Art Generator

An interactive web application for creating, customizing, and exporting beautiful gradient backgrounds.

## Features

- **Interactive Gradient Editor**: Add, remove, and customize color stops with real-time preview
- **Multiple Gradient Types**: Linear, radial, and conic gradients
- **Custom Controls**: Adjust angle, colors, and positions with intuitive sliders
- **Preset Gallery**: Choose from professionally designed gradient presets
- **Export Options**: 
  - Copy CSS code with one click
  - Download gradient as PNG image
- **Random Generator**: Get inspired with randomly generated gradients
- **Responsive Design**: Works on desktop and mobile devices

## How to Use

1. **Customize Gradient Type**
   - Select between linear, radial, or conic gradients using the type buttons
   
2. **Adjust Gradient Angle** (for linear and conic)
   - Use the angle slider or directly input the angle value
   
3. **Manage Color Stops**
   - Click on color boxes to change colors
   - Adjust position with sliders
   - Add new color stops with the "+ Add Color Stop" button
   - Remove color stops (minimum 2 required)
   
4. **Use Presets**
   - Click on any preset to apply it instantly
   - Use as a starting point for your own designs
   
5. **Export Your Creation**
   - Copy CSS code to use in your projects
   - Download as a high-quality PNG image

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- Vanilla JavaScript
- Canvas API for image export
- Font Awesome for icons
- Google Fonts (Poppins and Roboto Mono)
