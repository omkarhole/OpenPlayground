About Screen Shaking

Screen shaking effects are used in various applications:

  • Game Development: To simulate impacts, earthquakes, or explosions
  • UI/UX Testing: To test interface stability under simulated device movement
  • Accessibility: To understand how motion affects users with vestibular disorders
  • Education: To demonstrate earthquake intensity and effects

How It Works

This simulator uses CSS animations and JavaScript to create realistic screen shaking effects:

  • Intensity: Controls how far the screen moves during shaking
  • Duration: How long the shaking effect lasts
  • Frequency: How fast the screen moves back and forth
  • Type: Different patterns of movement for various scenarios

At high intensities, visual damage effects like cracks and red overlays appear.
