# Interactive Periodic Table with Element Quiz

A complete interactive web-based periodic table featuring all 118 known chemical elements with detailed information and an educational quiz mode.

## Features

1. **Complete Periodic Table**
   - All 118 chemical elements with accurate positioning
   - Color-coded by element categories
   - Proper separation of lanthanides and actinides
   - Responsive design that works on desktop and mobile

2. **Detailed Element Information**
   - Atomic number, mass, and symbol
   - Period and group information
   - Electron configuration (simplified)
   - State at room temperature
   - Year of discovery
   - Density and melting point (for many elements)
   - Element category with color coding

3. **Interactive Features**
   - Click on any element to view detailed information
   - Smooth animations and hover effects
   - Placeholder indicators for lanthanides/actinides that scroll to the correct row
   - Two modes: View Mode and Quiz Mode

4. **Educational Quiz Mode**
   - 4 types of questions: symbol, name, atomic number, and category
   - Questions generated randomly from all 118 elements
   - Instant feedback with correct/incorrect highlighting
   - Score tracking and progress monitoring
   - Next question button to continue learning

5. **Visual Design**
   - Modern dark theme with color-coded elements
   - Clear legend showing all element categories
   - Responsive layout that adapts to screen size
   - Smooth transitions and interactive feedback

## Element Categories

The periodic table color-codes elements into 10 categories:
- **Alkali Metals** - Red
- **Alkaline Earth Metals** - Orange
- **Transition Metals** - Blue
- **Basic Metals** - Green
- **Semimetals** - Purple
- **Nonmetals** - Light Blue
- **Halogens** - Yellow
- **Noble Gases** - Deep Purple
- **Lanthanides** - Pink
- **Actinides** - Brown

## How to Use

1. **View Mode (Default)**
   - Click on any element in the periodic table to see its details
   - Information appears in the right panel
   - Includes atomic properties, discovery information, and physical characteristics

2. **Quiz Mode**
   - Click the "Quiz Mode" button to start the quiz
   - Answer questions about elements (symbol, name, atomic number, or category)
   - Click on your answer choice
   - See instant feedback with correct answer highlighted
   - Click "Next Question" to continue
   - Track your score in real-time

3. **Navigation**
   - Use the "View Mode" button to return to element exploration
   - Click "Reset" to clear quiz scores and return to default view
   - Click on the * placeholder in the main table (La or Ac) to scroll to lanthanides/actinides
