What is a Finite Automaton?

A finite automaton (FA) is a mathematical model of computation used to recognize patterns within input strings. It consists of states, transitions between states, and acceptance criteria.

Types of FA:

  • DFA (Deterministic Finite Automaton): Each state has exactly one transition for each input symbol.
  • NFA (Nondeterministic Finite Automaton): States can have multiple transitions for the same input or epsilon transitions.

Components:

  • States: Represented as circles on the canvas.
  • Transitions: Arrows between states labeled with input symbols.
  • Start State: Marked with an arrow pointing to it.
  • Accept States: Marked with double circles.

How to Use:

  1. Add states by clicking "Add State" and clicking on the canvas.
  2. Add transitions by clicking "Add Transition" and selecting two states.
  3. Set start and accept states using the respective buttons.
  4. Enter a string and click "Simulate" to see step-by-step processing.

Applications:

  • Pattern matching in text editors
  • Compiler design (lexical analysis)
  • Network protocol validation
  • Regular expression engines
