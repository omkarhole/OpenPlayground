A simple Time-Locked Notes application built using HTML, CSS and Vanilla JavaScript.

## Features
- Create messages for the future
- Messages unlock automatically at chosen time
- Data stored in localStorage
- Clean and responsive UI

## Tech Stack
- HTML
- CSS
- JavaScript

## How It Works
Each message is saved with an unlock timestamp.  
On every render, current time is compared with unlock time.

## Run Locally
Open index.html in browser.
