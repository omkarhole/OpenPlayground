BST Concepts

What is a Binary Search Tree?

A Binary Search Tree (BST) is a data structure where each node has at most two children, and for each node, all elements in its left subtree are less than the node, and all elements in its right subtree are greater than the node.

Time Complexities

  • Search: O(h) where h is the height of the tree (O(log n) for balanced trees)
  • Insert: O(h) (O(log n) for balanced trees)
  • Delete: O(h) (O(log n) for balanced trees)
  • Traversal: O(n) for all nodes

Traversals

  • Inorder: Left subtree, root, right subtree (sorted order)
  • Preorder: Root, left subtree, right subtree
  • Postorder: Left subtree, right subtree, root

Use Cases

  • Searching and sorting data efficiently
  • Implementing associative arrays (like maps in programming)
  • Database indexing
  • Priority queues and heaps
