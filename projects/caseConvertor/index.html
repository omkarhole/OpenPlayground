About Naming Conventions

camelCase

First word lowercase, subsequent words capitalized. Used in JavaScript for variables and functions.

exampleVariableName

PascalCase

Each word capitalized. Used for class names in many programming languages.

ExampleClassName

snake_case

All lowercase with underscores between words. Common in Python and databases.

example_variable_name

kebab-case

All lowercase with hyphens between words. Used in URLs and CSS classes.

example-variable-name

CONSTANT_CASE

All uppercase with underscores between words. Used for constants in many languages.

EXAMPLE_CONSTANT_NAME
