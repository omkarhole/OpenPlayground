# Push Da Box

A retro-styled HTML5 puzzle game inspired by Sokoban. Push orange boxes onto target spots while navigating through increasingly complex levels. Features a Game Boy-inspired interface with chiptune audio.

## How to Play

- **Objective:** Push all orange boxes ($) onto the target spots (.) to clear each sector
- **Controls:**
  - Arrow Keys or WASD: Move the player (@)
  - Z: Undo last move
  - R: Reset current level
  - Touch controls available on mobile

- **Game Elements:**
  - @: Player character
  - $: Orange boxes to push
  - .: Target spots
  - #: Walls (impassable)

## Features

- 5 challenging levels
- Move counter and level progress tracking
- Undo and reset functionality
- Retro Game Boy aesthetic
- Built-in audio system with chiptune sounds
- Mobile touch support
- Fullscreen mode

## Files

- `index.html`: Main game interface
- `app.js`: Game logic, level data, and audio system
- `style.css`: Retro Game Boy styling

## Running the Game

Open `index.html` in a modern web browser. Click "INITIALIZE" to start the game.

## Technologies Used

- HTML5 and CSS3 for interface
- Vanilla JavaScript for game mechanics
- Web Audio API for sound generation
