Regex Cheatsheet

Character Classes

  • . - Any character except newline
  • \d - Digit (0-9)
  • \w - Word character (a-z, A-Z, 0-9, _)
  • \s - Whitespace (space, tab, newline)
  • [abc] - Any of a, b, or c
  • [^abc] - Not a, b, or c

Quantifiers

  • * - 0 or more
  • + - 1 or more
  • ? - 0 or 1
  • {3} - Exactly 3
  • {3,5} - Between 3 and 5
  • {3,} - 3 or more

Anchors

  • ^ - Start of string
  • $ - End of string
  • \b - Word boundary
  • \B - Not word boundary

Common Patterns

  • ^\d{5}$ - 5 digit zip code
  • ^\w+@\w+\.\w+$ - Simple email
  • ^https?:// - HTTP/HTTPS URL
  • ^\d{3}-\d{3}-\d{4}$ - US phone
