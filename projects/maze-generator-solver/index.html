Algorithm Info

Generation Algorithms

Recursive Backtracking

Starts at a random cell and recursively visits unvisited neighbors, carving paths. Creates long, winding mazes with only one solution path between any two points.

Prim's Algorithm

Creates a maze by randomly adding walls and connections like growing a tree. Produces more branching paths than recursive backtracking.

Kruskal's Algorithm

Randomly selects edges and adds them if they connect different components. Creates a perfect maze with multiple possible paths.

Solving Algorithms

BFS (Breadth-First Search)

Explores all neighbors at current distance before moving to next level. Guarantees shortest path in unweighted grids.

DFS (Depth-First Search)

Explores as far as possible along each branch before backtracking. Faster but may not find shortest path.

A* Algorithm

Uses heuristics to find shortest path efficiently. Combines actual cost from start with estimated cost to end.

Dijkstra's Algorithm

Similar to A* but without heuristics. Guarantees shortest path by exploring all possible routes.
