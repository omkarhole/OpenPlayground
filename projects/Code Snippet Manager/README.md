# Code Snippet Manager

A web tool to save, organize, and search code snippets with tags, language detection, and syntax highlighting. Users can quickly copy, edit, and categorize their favorite code blocks.

## Features
- Add/edit/delete code snippets
- Organize snippets by tags and programming language
- Syntax highlighting for multiple languages
- Search and filter snippets
- Copy to clipboard with one click
- Import/export snippets as JSON

## Usage
1. Open `index.html` in your browser.
2. Add, edit, or delete code snippets.
3. Organize by tags and language.
4. Search and filter snippets.
5. Import/export your snippets as JSON.

## Tech
- Vanilla HTML, CSS, JavaScript

## Responsive
- Fully responsive for desktop and mobile.
