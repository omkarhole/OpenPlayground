# Cosmic Clock

A mesmerizing HTML5 canvas visualization that represents time as concentric rings, each representing different time scales from seconds to years. The clock displays current time with smooth animations and includes a timeline of significant 2025 events.

## Features

- **Concentric Time Rings:**
  - Inner rings: Seconds, Minutes, Hours (fast-moving)
  - Middle rings: Day of Week, Week, Month
  - Outer rings: Quarter, Season, Year (slow-moving)

- **Interactive Elements:**
  - Hover tooltips showing exact values
  - Time speed controls (1x, 1m/s, 1h/s, 1d/s)
  - Event filtering by category

- **2025 Events Timeline:**
  - Political, Sports, Entertainment, Cultural, and Astronomical events
  - Filterable event panel
  - Hover to highlight events on the clock

- **Visual Design:**
  - Smooth animations and gradients
  - Color-coded rings with legends
  - Responsive canvas that fills the screen

## How to Use

1. Open `index.html` in a modern web browser
2. Watch the rings rotate to show current time
3. Hover over rings for detailed information
4. Use speed controls to accelerate time
5. Browse and filter 2025 events in the side panel

## Ring Legend

- **Red:** Seconds
- **Orange:** Minutes
- **Cyan:** Hours
- **Green:** Day of Week
- **Purple:** Week
- **Pink:** Month
- **Blue:** Quarter
- **Teal:** Season
- **Orange:** Year

## Files

- `index.html`: Main HTML structure and UI
- `app.js`: Canvas rendering, time calculations, and event data
- `style.css`: Styling for UI elements and layout

## Technologies Used

- HTML5 Canvas for rendering
- JavaScript for animations and time calculations
- CSS for responsive design
