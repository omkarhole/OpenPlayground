How to Use Gesture Controls
Press and hold left mouse button, then draw a gesture shape in the canvas area.
Draw recognizable shapes like circles, squares, arrows, or letters to trigger actions.
Each gesture triggers a different action - control volume, brightness, media playback, and more.
Adjust sensitivity and other settings in the control panel for optimal recognition.
